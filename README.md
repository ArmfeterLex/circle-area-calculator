# Circle Area Calculator

Простое приложение на C# Windows Forms для вычисления площади круга по радиусу.

## Функциональность

- Выбор радиуса через TrackBar
- Автоматический расчёт площади круга: S = π × R²
- Отображение радиуса и площади в текстовых полях
- Мгновенное обновление при перемещении ползунка

## Интерфейс

- trackBar1 — ползунок выбора радиуса
- textBox1 — отображение радиуса
- textBox2 — отображение площади

## Пример работы

Радиус: 5
Площадь: 78.54

## Куда можно встроить

- В геометрические калькуляторы
- В инженерные расчёты
- В демонстрацию работы TrackBar
- В обучающие проекты

## Требования

- Windows
- .NET Framework / .NET Core с WinForms

## Примечание

При перемещении ползунка радиус отображается целым числом. Для более точных значений можно изменить шаг TrackBar.
